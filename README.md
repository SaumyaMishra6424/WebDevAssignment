 🌐 Responsive Web Page Assignment

This project is a fully responsive web page built using **vanilla HTML, CSS, and JavaScript** (no frameworks or libraries), following the provided Figma design specifications.

---

 Project Overview

The goal of this assignment is to create a pixel-perfect, responsive webpage with interactive UI components, including:

- Sticky header with scroll behavior
- Image carousel with zoom functionality
- Fully responsive layout across devices

---

🎯 Features Implemented

 1. Pixel-Perfect Design
- Followed the Figma design precisely
- Maintained spacing, typography, and layout consistency

---

 2. Responsive Design
- Optimized for:
  - Desktop 💻
  - Tablet 📱
  - Mobile 📲
- Used **Flexbox & CSS Grid**
- Media queries for adaptive layouts

---

3. Sticky Header
- Appears after scrolling past the first fold
- Positioned above the navigation bar
- Smooth show/hide transitions
- Hides when scrolling back up

---

 4. Image Carousel with Zoom
- Interactive image slider
- Hover-based zoom preview effect
- Smooth animations and transitions
- Matches Figma behavior

---

 5. Clean Code Structure
- Semantic HTML5 elements
- Modular CSS
- Well-commented JavaScript
- Organized file structure
